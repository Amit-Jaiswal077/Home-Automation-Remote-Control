# Home-Automation-Remote-Control
“A simple Arduino‑based home automation system that allows users to control appliances remotely using an IR remote. The project demonstrates switching logic using a relay/transistor, enabling ON/OFF control of lights or fans. Easy to build, extendable, and beginner‑friendly.”
# Components Used:
1. Arduino Uno R3
2. IR sensor
3. Relay SPDT
4. Light bulb
5. 5V , 5A Power Supply
